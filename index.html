Optimizing Travel Routes

Developed a model that provides an optimal economic path for traveling between multiple cities in real-time. The model takes the names of the desired cities as input and utilizes Google API to obtain distance matrices between the cities. We implemented the model using pymo in Python. By leveraging the distance matrices and incorporating optimization techniques, our model determines the most efficient route that minimizes travel costs. The end result is a calculated itinerary that outlines the recommended order in which to visit the cities, ensuring an economically efficient travel plan.

Clusture analysis on wine dataset

In this analysis, we used wine quality data to perform clustering and dimensionality reduction. We identified 8 clusters as optimal based on a "Within Sum-of-Square" analysis. Additionally, Principal Component Analysis (PCA) revealed that 6 principal components explained approximately 80% of the dataset's variance. Cluster analysis aimed to minimize clusters while fully explaining the data, while PCA sought to minimize principal components while capturing dataset variation. These findings guide our data segmentation and dimensionality reduction strategies.
